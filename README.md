# 🍕 Pizzería Mamma Mía - Hito 2

Proyecto desarrollado con React + Vite como parte del desafío de Desafío Latam.

---

## 🚀 Descripción

Crea un componente Register que contenga un formulario con los siguientes campos:
● Email
● Contraseña
● Confirmar contraseña
